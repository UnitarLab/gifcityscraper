# gifcityscraper

Retro-styled vintage animated GIF collector for the GeoCities Archive (via [GifCities](https://gifcities.org/)).

## Features
- ✨ Cyberpunk/Retro UI with scanline effects.
- ⚡ Multiple search categories (Elements, Animals, Plants, etc.).
- ＋ Custom term support.
- ⬇ Batch download as ZIP or export as TXT/CSV.
- ◈ Live gallery preview.

## Deployment
Optimized for Vercel. Simply push to GitHub and connect yours.
